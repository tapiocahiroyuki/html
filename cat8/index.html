グループ

  • 投稿日:2018年2月26日
  • by takeoka
  • Category: フォーム
input要素にテキストやボタンを追加して、グループにできます。textareaやselect要素には使えません。label要素やaria-label系の属性を使用し、スクリーンリーダーへ配慮してください。
@
@example.com
$ .00
https://example.com/users/ 

<div class="input-group">
<span class="input-group-addon" id="basic-addon1">@</span>
<input type="text" class="form-control" placeholder="Username" aria-describedby="basic-addon1">
</div>
<div class="input-group">
<input type="text" class="form-control" placeholder="Recipient's username" aria-describedby="basic-addon2">
<span class="input-group-addon" id="basic-addon2">@example.com</span>
</div>
<div class="input-group">
<span class="input-group-addon">$</span>
<input type="text" class="form-control" aria-label="Amount (to the nearest dollar)">
<span class="input-group-addon">.00</span>
</div>
<label for="basic-url">Your vanity URL</label>
<div class="input-group">
<span class="input-group-addon" id="basic-addon3">https://example.com/users/</span>
<input type="text" class="form-control" id="basic-url" aria-describedby="basic-addon3">
</div>

</div>
</div>
<div class="form-group form-group-sm">
<label class="col-sm-2 control-label" for="formGroupInputSmall">ラベル小</label>

<input class="form-control" type="text" id="formGroupInputSmall" placeholder="Small input">
</div>
</div>
</form>

ラベルが隠しテキストのアイコン付きフォーム

  • 投稿日:2018年2月22日
  • by takeoka
  • Category: フォーム
ラベルがないフォームでも、label.sr-onlyでラベル名を読み上げられるようにしてください。
(success) 

<div class="form-group has-success has-feedback">
<label class="control-label sr-only" for="inputSuccess5">快特の停車駅を入力</label>
<input type="text" class="form-control" id="inputSuccess5" aria-describedby="inputSuccess5Status">
<span class="glyphicon glyphicon-ok form-control-feedback" aria-hidden="true"></span>
<span id="inputSuccess5Status" class="sr-only">(success)</span>
</div>
